## Markdown guide

## Headings
```
# H1
## H2
### H3
#### H4
##### H5
###### H6
```

## Emphasis
```
*italic* or _italic_
**bold** or __bold__
***bold italic*** or ___bold italic___
```

## Lists
### Unordered List
```
- Item 1
- Item 2
  - Subitem 2.1
* Item 3
```

### Ordered List
```
1. First
2. Second
3. Third
   1. Subitem 3.1
``` 

## Links
```
[Link text](https://example.com)
```

## Images
```
![Alt text](https://via.placeholder.com/150)
```

## Blockquotes
```
> This is a blockquote.
>
> It can span multiple lines.
```

## Inline Code
```
`inline code`
```

## Code Blocks
```python
def hello():
    print("Hello, World!")
```

## Horizontal Rule
```
---
```

## Tables
```
| Header 1 | Header 2 |
|----------|----------|
| Row 1    | Data     |
| Row 2    | Data     |
```

## Task Lists
```
- [x] Completed task
- [ ] Incomplete task
```

## Strikethrough
```
~~strikethrough~~
```

## Emoji
```
:smile: :rocket: :tada:
```
